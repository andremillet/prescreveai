Bem-vindo ao PrescreveAI!

O PrescreveAI é uma ferramenta de linha de comando e API para auxiliar na geração de prescrições médicas, permitindo a criação de documentos em PDF e a integração com outros sistemas.

Como Instalar

Para instalar o PrescreveAI em seu sistema e tê-lo disponível globalmente, abra seu terminal e execute o seguinte comando:

Este comando irá baixar e executar o script de instalação, que configurará o ambiente necessário e criará o comando prescreveai em seu sistema.

Após a Instalação

Você poderá usar o comando prescreveai para interagir com a ferramenta:

  • Para usar a interface de linha de comando interativa:
    • prescreveai
  • Para iniciar o servidor da API (para integração com outros programas):
    • prescreveai serve

Para mais detalhes e documentação, visite o repositório no GitHub.
